# Adorsholipi
An android app for kinds under age of 5 to learn Arabic, Banlga &amp; English alphabets. 

Kids can learn the following feature using this application with proper sound:

1. Bangla:
 - Bangla alphabet (Vowels and consonants)
 - Number
 - Day name
 - Month name
 - Season name
 
2. English: 
 - Alphabet
 - Number
 - Day name
 - Month name
 - Season name
 
3. Arabic: 
 - Alphabet
 - Number
 - Day name
 - Month name
 - Season name
